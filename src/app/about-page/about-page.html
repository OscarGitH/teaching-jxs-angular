À propos du Pokédex Angular

Ce projet est réalisé dans le cadre d'un TP Angular afin de découvrir le framework et ses concepts clés. L'objectif principal est de créer un Pokédex interactif, modulaire et réutilisable, basé sur Angular et Typescript.

Objectifs du projet

  • Apprendre à structurer une application Angular avec des composants et services.
  • Utiliser le data-binding pour lier les vues et le modèle.
  • Intégrer des API externes pour récupérer des données de Pokémon.
  • Mettre en place un filtrage et une pagination pour naviguer dans la liste des Pokémon.
  • Découvrir les bonnes pratiques de modularité et de réutilisabilité dans Angular.
  • Expérimenter avec Angular Material pour améliorer l'interface utilisateur.
